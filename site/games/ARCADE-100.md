## ART NEEDED FROM USER

Most games use PROCEDURAL drawing (like existing tunnel-bug, spore-jumpers). But for extra polish:

### Priority Sprites (reusable across many games)
1. **Baseling hero** (idle, run-left, run-right, jump, attack) — 16x16 or 32x32
2. **Baseling enemy variants** (slime, spore, mushroom, crystal) — 16x16
3. **Token collectibles** (MfT coin, POOP pellet, food item, LP gem) — 8x8 or 16x16
4. **Tileset: dirt/stone/grass/lava** — 16x16 tiles
5. **Boss sprites** (bigger versions, 32x32 or 48x48)
6. **Vehicle** (boat, car, spaceship) — 24x24

### Optional (nice-to-have, not blocking)
- Background parallax layers (sky, mountains, caves)
- Title screen logos per game
- Character portraits for RPG dialogue
- World map tiles (for dungeon-crawler/RPG games)

### NOT NEEDED (I'll draw procedurally)
- Basic shapes (platforms, walls, projectiles)
- UI elements (health bars, score text, menus)
- Particle effects (explosions, sparkles)
- Simple enemy patterns (geometric shapes with eyes)

## BUILD ORDER
1. Start with Batch 2 (puzzles) — fastest to build, most replayable
2. Then Batch 1 (platformers) — crowd pleasers
3. Then Batch 3 (shooters) — simple loop, satisfying
4. Mix in Batch 4-7 as we go
