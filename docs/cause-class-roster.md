# Cause = Class Roster — FIRST-PASS DRAFT

**Status:** DRAFT for the founder to refine. The founder owns final creative — every
class name, stat pick, prereq, and strictness band below is a PROPOSAL to react to,
not a decision. Assumptions and placeholders are flagged inline.

**Skeleton used:** `MfT-Launch/docs/battle-grid-class-map.md` (the web-verified D&D
5e/3.5 + FFT job tree + 16 DMG prestige class map, the 11 archetype families, the
universal Tier 0→3 ladder, the 3 gating levers: breadth / depth / conduct-identity).

**Game rules this rides on** (from `project_battle_grid`):
- 1 level = $1 permanently endowed. Stats: base 10 each, $1 = 1 point.
- **BOUGHT** regular water = diffuse 1/6 across all stats (bland). **EARNED** cause-water
  = full point on that cause's stat (6× concentrated → power + identity).
- Class = a LIVE function of **cause % of total endowment** (the concentration RATIO).
  Dilute below the line → **DRIFT** (class sleeps, never lost; re-concentrate to wake).
- **STRICTNESS** = required-to-free stat ratio + how tight the % band is. Loose (Fighter)
  → strict (deep exotics that drift the instant ratios slip).
- Every endowment funds trees (the floor); the "+cause" is the build.

## C. Tiered roster

Strictness scale: **Loose** (wide band, hard to drift) → **Medium** → **Strict** →
**Very strict** (narrow multi-band, drifts the instant a ratio slips).

### Tier 3 — Strict high-investment EXOTICS (broad + deep + very tight bands)

Require **4+ causes held above threshold within narrow concentration bands** (Mime's six
prereqs; Archmage's 15/15). Highest power, hardest to maintain — drops to a Tier-2 fallback
the instant a band slips. The strictest classes = the most ONGOING permanent charity (the
mission payoff: rarest builds demand sustained concentrated giving).
